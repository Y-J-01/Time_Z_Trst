# 🧟 Time_Z_Trst

## 📌 Overview
**Time_Z_Trst** is an early Unreal Engine project focused on building core gameplay systems such as character control, UI, animation, and basic level design.

This project represents one of my first steps into gameplay programming in Unreal Engine, where I explored how to create a playable character, connect UI elements, work with animation systems, and experiment with environment creation.

---

## 🎯 Purpose
The main goal of this project was to:
- Learn how Unreal Engine handles **character systems**
- Understand **GameMode, HUD, and UI workflow**
- Connect **C++ gameplay logic with Blueprint UI**
- Work with **animation instances**
- Explore **level design basics** (object placement, environment setup)

---

## ⚙️ Features

### 👤 Character System
- Custom character class: `AMyCharacter`
- Movement and input system
- Configurable character parameters (speed, jump, etc.)
- Camera setup and player control

---

### 🎮 Game Mode
- Custom GameMode: `AMainCharachterGameMode`
- Controls game rules and player setup

---

### 🖥️ UI System
- Custom HUD: `ACharacterHUD`
- UI widgets implemented with UMG:
  - `UUserWidgetCardio` (cardio/stamina-like system)
- Connection between character data and UI

---

### 🎞️ Animation System
- Animation instance: `UCharacterAnimInstanse`
- Integration between character movement and animation

---

### 🏞️ Level Design (Beginner Exploration)
- Manual placement of environment objects:
  - Houses
  - Trees
  - Interior elements
  - Terrain elements (mountains, landscape)
- Basic scene composition and layout
- First experiments with creating a believable environment inside Unreal Engine

---

## 🧠 What I Learned
- Unreal Engine C++ architecture (Actors, Components, GameMode)
- How to bind **input → character → animation**
- Creating and connecting **HUD + Widgets**
- Using **UPROPERTY / UFUNCTION** for Blueprint integration
- Basics of **level design and environment setup**
- Working with assets and scene composition

---

## 🛠️ Tech Stack
- Unreal Engine 5.6
- C++
- UMG (UI system)
- Animation Blueprint system
